### Overview
    A documentation on learning on intermediate-level foundation in networking design and operations, including hierarchical network structures, virtualization, and cloud services.

## Learning Objectives
    Develop practical math skills for working across decimal, binary, and hexadecimal systems, and gain a deep understanding of how Ethernet, ARP, DNS, DHCP, and transport layer protocols function within a network.

## Topics Covered
    Module 1: Network Design
    Module 2: Cloud and Virtualization
    Module 3: Number Systems
    Module 4: Ethernet Switching
    Module 5: Network Layer
    Module 6: IPv4 Address Structure
    Module 7: Address Resolution
    Module 8: IP Addressing Services
    Module 9: Transport Layer
    Module 10: CISCO IOS Command Line
    Module 11: Build a Small Cisco Network
    Module 12: ICMP
